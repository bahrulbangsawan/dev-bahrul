# Setup Guide

Complete guide to setting up Project 1.

## Prerequisites

- Node.js 18+
- Git
- Text editor

## Installation

<Steps>

1. **Clone the repository**
   ```bash
   git clone https://github.com/example/project-1.git
   cd project-1
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   Visit `http://localhost:3000`

</Steps>

<Aside type="tip">
This is part of a nested sidebar structure demo. You can create as many levels as you need!
</Aside>

## Configuration

Edit `config.js` to customize your setup:

```javascript
export default {
  name: "Project 1",
  version: "1.0.0",
  port: 3000
}
```
