# Project 1 Overview

This is an example project showcasing **nested sidebar groups** in Starlight.

## About This Project

Project 1 demonstrates how to structure documentation with multiple levels of navigation:

- Parent Group: **Projects**
- Sub-group: **Project 1**
- Pages: Overview, Setup Guide, Deployment

## Features

- Clean nested navigation
- Collapsible sidebar groups
- Easy to organize large documentation sets
